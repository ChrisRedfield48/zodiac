## 📖 О проекте

Пользователь выбирает месяц и вводит день рождения — приложение определяет знак зодиака и выводит его символ, стихию и описание характера.

## ⚙️ Как это работает

### Данные — массив `myZodiac`

Хранит 12 объектов, по одному на каждый знак:

```js
{
  name: 'Овен',
  symbol: '♈️',
  element: 'Огонь',
  startMonth: 3, startDay: 21,
  endMonth: 4,   endDay: 19,
  desc: 'Энергичный, смелый и инициативный...'
}
```

### Логика — функция `search()`

```
1. Считать месяц и день из полей ввода
2. Валидация: поля не пустые, день в диапазоне 1–31
3. forEach по массиву myZodiac:
   если (месяц === startMonth && день >= startDay)
   или   (месяц === endMonth   && день <= endDay)
   → найден знак
4. Показать блок с результатом
```

## ✅ Валидация

| Ситуация | Сообщение об ошибке |
|----------|---------------------|
| Поля не заполнены | `Строки не должны быть пустые!` |
| День < 1 или > 31 | `День должен быть от 1 до 31` |
